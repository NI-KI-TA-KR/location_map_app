# Location Map App

Тестовий застосунок з використанням Flutter Map

## Опис

Тестовий застосунок з використанням Flutter Map

### Можливості:

- Відображення карти з кастомними маркерами.
- Зміна теми карти (темна або світла).
- Кластеризація маркерів.
- Підтримка мультимедійних даних: додавання інформації про точку (назва, опис).
- Інтерактивність: натискання на маркер відкриває картку з подробицями.



## Технологии

Приложение использует следующие технологии:

- **Flutter** — для створення кросплатформеного мобільного додатку.
- **flutter_map** — для інтеграції карт.
- **flutter_map_marker_cluster** — для кластеризації маркерів.
- **flutter_bloc** — для керування станом за допомогою BLoC.
- **MapTiler API** — для надання карт з різними стилями (темна, світла, гібридна тощо).
- **sqlite3** — для роботи з базою даних SQLite.
- **drift** — для роботи з локальними базами даних з додатковими можливостями ORM.
- **slang** — для роботи з базами даних, орієнтуючись на типи даних в Dart, і надання типів для SQL запитів.
